# 智能飞镖感应设备测试项目

## 项目简介

本项目是针对CST2025全国大学生软件测试大赛嵌入式分赛省赛的智能飞镖感应设备测试代码实现。

## 测试内容概览

### 功能测试

1. **1shoot.lua - 飞镖信息输入功能测试**
   - 测试轻镖和重镖的正常输入
   - 测试飞镖数量边界值
   - 测试异常输入处理

2. **2target.lua - 靶区信息接收功能测试**
   - 测试所有有效靶区（内外牛眼、三倍、双倍、单倍区域）
   - 测试无效靶区值的处理

### 接口测试

3. **3interface_shoot.lua - 数据传送机接口容错测试**
   - 包头错误处理测试
   - 数据标志错误处理测试
   - 包尾错误处理测试
   - 校验和错误处理测试
   - 多字段错误处理测试
   - 系统恢复能力测试

4. **4interface_target.lua - 靶体接口容错测试**
   - 包头错误处理测试
   - 数据标志1、2错误处理测试
   - 包尾错误处理测试
   - 校验和错误处理测试
   - 投中区域值错误处理测试
   - 系统恢复能力测试

## 快速开始

1. 确保测试环境已正确配置（详见 `project/env/env.env`）
2. 连接被测设备到指定的COM端口
3. 按顺序运行测试：
   - 先运行功能测试：1shoot.run → 2target.run
   - 再运行接口测试：3interface_shoot.run → 4interface_target.run
4. 根据测试框架的提示进行确认操作
5. 查看测试报告，记录发现的问题

## 技术要点

- **通信协议**：串口通信（RS-232）
- **波特率**：9600
- **数据位**：8位
- **奇偶校验**：无
- **停止位**：1位
- **测试脚本语言**：Lua

## 开发说明

- 协议定义在 `project/prot/` 目录
